# Vendor Compliance Addendum

Document owner: Compliance Operations  
Effective date: 2026-04-01  
Status: Current

## A. Evidence handling

A.1 Incident response plans are reviewed during onboarding and during periodic vendor reassessment.

A.2 The review confirms whether a plan exists and whether required sections are present. It does not set a deadline for maintaining the incident response plan.

## B. Interpretation

B.1 If this addendum conflicts with the Vendor Information Security Requirements, the current security requirements control.
