# Vendor Intake Mapping Addendum

Document owner: Compliance Operations  
Effective date: 2026-04-01  
Status: Current

## A. Extraction record shape

A.1 Downstream intake systems require every extracted requirement record to include the fields `obligation`, `actor`, `condition`, and `deadline`.

A.2 The deadline key must be present for every record. When the current policy does not state a deadline, the deadline value must remain null.

## B. Interpretation

B.1 This addendum describes the record shape for intake processing. It does not create new vendor obligations and does not authorize copying a deadline from another clause.
