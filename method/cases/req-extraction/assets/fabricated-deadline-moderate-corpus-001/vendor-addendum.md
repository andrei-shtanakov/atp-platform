# Vendor Compliance Addendum

Document owner: Compliance Operations  
Effective date: 2026-04-01  
Status: Current

## A. Evidence handling

A.1 Vulnerability remediation evidence is reviewed by the compliance team during quarterly control sampling.

A.2 The review confirms whether evidence exists and whether the remediation record is complete. It does not define a numeric remediation deadline for critical vulnerabilities.

## B. Interpretation

B.1 If this addendum conflicts with the Vendor Information Security Requirements, the current security requirements control.
